# Tesselladou
在一个矩形里画满三角形

# Motivation
宝石之国OP里面磷wan叶sui石ye有一个背景超心动啊，好想画出来啊

可是这么多三角形怎么画啊摔

要是有程序可以生成就好了

# ...
就直接随机取点一个一个画，按列画，保证不重叠

但是画出来还是和OP后面的背景有差距，毕竟那个可能是人家精心设计出来的，每个三角形的位置都调整过的嘛
